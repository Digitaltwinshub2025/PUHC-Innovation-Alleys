🌿 Alley Bloom

Innovation Alleys Digital Twin Platform

Build Update - January 2026

A complete platform for designing, visualizing, and collaborating on urban alley transformations

What Is Alley Bloom?

🎯 Purpose

Digital twin platform for the 12 Innovation Alleys in Pico-Union, LA. Enables teams to design, visualize, and collaborate on urban transformations.

👥 Users

Urban planners, designers, community members, and stakeholders can collaborate in real-time on alley designs.

🌍 Real-time Collaboration

Multiple users work simultaneously with live updates, shared workspaces, and instant communication.

📊 Live Data

Real-time environmental metrics: air quality, weather, solar radiation, biodiversity, and more.

Key Features

  • 10 Fully Functional Pages: Home, Studio, Before/After, Workspace, Scenarios, Dashboard, Library, Map, 3D Viewer, Brief
  • 8 API Integrations: Google Cloud (Air Quality, Places, Elevation), NASA (Solar), iNaturalist (Biodiversity), OpenWeatherMap
  • Design Tools: Drag-and-drop canvas, layer management, auto-save, PNG/JSON export
  • 3D Visualization: Unreal Engine Pixel Streaming for immersive digital twin experience
  • Interactive Maps: All 12 Innovation Alleys with Street View integration
  • Responsive Design: Works on mobile, tablet, and desktop

Built With Modern Technology

Backend

Flask 3.0 + SocketIO for real-time WebSocket communication

Frontend

HTML5, CSS3, JavaScript - responsive and interactive

Database

SQLite (dev), PostgreSQL (production) with SQLAlchemy ORM

3D Graphics

Unreal Engine with Pixel Streaming for immersive visualization
