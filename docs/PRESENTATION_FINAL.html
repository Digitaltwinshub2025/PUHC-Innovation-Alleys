🌿 Alley Bloom

Innovation Alleys Digital Twin Platform
• Complete platform for designing and visualizing alley transformations
• Real-time collaboration for teams
• Live environmental data integration

What Is Alley Bloom?

🎯 Purpose

Digital twin platform for 12 Innovation Alleys in Pico-Union. Enables teams to design and visualize alley transformations.

👥 Users

Urban planners, designers, community members, and stakeholders all collaborate in real-time on alley designs.

🌍 Real-time Collaboration

Multiple users work simultaneously on the same designs with live updates and shared workspaces.

📊 Live Data

Real-time environmental metrics: air quality, weather, solar radiation, biodiversity from 8 APIs.

Key Features

10 Fully Functional Pages
Home • Visualization Studio • Before & After • Design Workspace • Scenarios • Live Dashboard • Plant Library • Innovation Map • 3D Viewer • Design Brief
8 API Integrations
Google Cloud (Air Quality, Places, Elevation) • NASA (Solar Data) • iNaturalist (Biodiversity) • OpenWeatherMap (Weather)
Design Tools
Drag-and-drop canvas • Layer management • Auto-save • Export to PNG/JSON
Advanced Features
3D Visualization (Unreal Engine) • Interactive Maps (Street View) • Responsive Design (Mobile/Tablet/Desktop)

Technical Stack

Backend

Flask 3.0 + SocketIO for real-time WebSocket communication

Frontend

HTML5, CSS3, JavaScript - responsive and interactive

Database

SQLite (dev), PostgreSQL (production) with SQLAlchemy ORM

3D Graphics

Unreal Engine with Pixel Streaming for immersive visualization

Status: Production-ready, fully tested, comprehensive documentation included
