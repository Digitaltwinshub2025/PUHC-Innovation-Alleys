## PROJECT OVERVIEW

### What Is This Project?
The PUHC Innovation Alleys platform is a **digital twin visualization system** for transforming Water Alley in Pico-Union, Los Angeles. It demonstrates how underutilized urban alleys can become climate-resilient, community-centered spaces through three design interventions: Shade Structures, Community Murals, and Urban Farming.

### Core Technology
- **Platform:** Flask web application
- **3D Engine:** Unreal Engine 5 (photorealistic rendering)
- **Deployment:** Static site generator for GitHub Pages
- **Frontend:** HTML5, CSS3, JavaScript (no frameworks)
- **Styling:** Custom dark theme with cyan/teal accents
