## 1. DELETED FILES AND FOLDERS

### Major Directories Removed:

#### **A. `docs/` Folder** (Entire static build output - 380+ files)
**Reason:** This was a static HTML build generated by `build_static.py`. The current application runs on Flask, making this folder completely redundant.

**Contents Removed:**
- Static HTML pages (index.html, existing.html, compare.html, etc.)
- Duplicate CSS files (global-theme.css, style.css, etc.)
- Duplicate JavaScript files (main.js, scenarios.js, etc.)
- Duplicate image assets (400+ images)
- Duplicate medallion graphics (60+ files)
- Duplicate mural concepts (10+ files)
- Street view images (20+ files)
- Unreal Engine photos (7 files)
- Static video files (alley3-flythrough-compressed.mp4)
- Content JSON files (area-a.json, area-b.json, etc.)

#### **B. `server.js`** (Node.js development server)
**Reason:** This was a simple Node.js HTTP server for serving the old static build. The current application uses Flask (`app.py`) with Gunicorn for production, making this file obsolete.

**Routes in server.js (now handled by Flask):**
- `/visualization-studio` → Not in current build
- `/street-view-designer` → Not in current build
- `/before-after` → Not in current build
- `/design-workspace` → Not in current build
- `/design-brief` → Not in current build
- `/scenarios` → Not in current build
- `/live-dashboard` → Not in current build
- `/plant-library` → Not in current build

#### **C. `content/` Folder** (Empty directory)
**Reason:** Unused, no files inside. Content is now managed by `content_manager.py` and stored elsewhere.

#### **D. `data/` Folder** (Empty directory)
**Reason:** Unused, no files inside. Data is now managed by `data_manager.py` and SQLite database.
